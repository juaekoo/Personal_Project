1. 홈화면 (home)
- HTML : 상단메뉴와 로그인/회원가입 페이지 연결 (home.html)
- CSS : 가로 스크롤(Flex이용)과 포스터에 마우스 올릴 경우 밝기변화. 영화별로 포스터 크기가 달라도 포스터 비율변화 없이 화면에는 같은 사이즈로 출력되도록 함 (home.css)
- JavaScript : 영화데이터를 연결해 화면에 뿌리고, 영화포스터/영화명/별점 클릭하면 각 영화의 id가 달린 URL로 이동

2. 개봉예정 (upcoming)
- JavaScript : 개봉예정영화의 데이터(개봉일, 서비스업체) 정보를 받아오고 화면에 출력. 개봉예정일의 경우 연도나 월까지만 작성하면 자동으로 1월1일 또는 1일이 적용되기 때문에 해당부분 제거하고 출력
- CSS : Grid 이용
- 그 외 홈화면과 기능 동일

3. 컬렉션 (collection) 
- HTML : 표로 구성

4. 컬렉션만들기 (makeCollection) 
- CSS : Grid 이용
- JavaScript : CollectionData에 성공적으로 넘겼고(주석clg참고), 서버를 통해 다시 화면에 뿌리는 것은 백엔드에서 구현 예정

5. 로그인 (login)
- jQuery : 빈칸인지, 형식을 만족하는지 체크하고 유효성검사를 통과하지 못하면 박스 CSS색처리하고 알림문구 띄움

6. 회원가입 (signup)
- 로그인과 마찬가지로 jQuery로 구성
